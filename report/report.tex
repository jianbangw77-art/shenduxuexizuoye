\begin{abstract}
工业缺陷检测是计算机视觉在智能制造中的重要应用，其目标是在异常样本稀缺的条件下识别产品表面的划痕、污染、破损和结构异常。本文选择 DRAEM 的重建-判别联合思想作为 baseline，并使用 PyTorch 从零实现轻量化模型 DRAEM-Lite。该方法仅使用正常样本训练，通过合成异常构造伪像素标签，同时优化重建网络和判别分割网络。在此基础上，本文实现改进模型 DRAEM-Lite-MS，引入结构化异常合成、多尺度分割监督、Focal loss、Dice loss 以及重建残差融合。本文在 MVTec AD 数据集中的 bottle、cable 和 hazelnut 三个类别上进行实验。结果表明，改进方法在 bottle 和 cable 上取得明显提升，三类平均像素级 AUROC 从 0.628 提升到 0.704，平均 Best-F1 从 0.136 提升到 0.187，说明改进策略能够增强缺陷区域定位能力，但在 hazelnut 类别上仍存在泛化不足。
\end{abstract}

\noindent\textbf{关键词：} 缺陷检测；异常检测；DRAEM；MVTec AD；PyTorch

\section{引言}
工业视觉缺陷检测的任务是在产品表面图像中自动发现异常区域，例如瓶身污染、电缆破损、坚果裂纹等。与普通分类任务不同，工业场景中的异常样本通常数量少、形态复杂，并且真实生产过程中可能出现训练集中未见过的新型缺陷。因此，许多缺陷检测方法不依赖大量异常标注，而是将问题建模为异常检测：训练阶段主要使用正常样本，测试阶段输出图像级异常分数和像素级异常热力图。

MVTec AD 是工业异常检测领域常用的数据集，包含多种物体和纹理类别，并提供图像级标签和像素级缺陷掩码。当前代表性方法包括 PatchCore、DRAEM、SimpleNet 和 EfficientAD 等。其中，PatchCore 依赖预训练特征和记忆库检索，性能较强但实现流程较复杂；DRAEM 通过合成异常和判别分割网络完成端到端训练，更适合作为课程作业中的复现对象。本文选择 DRAEM 思路作为 baseline，从零实现模型、数据读取、损失函数、训练流程和评价指标，并进一步设计改进策略进行对比实验。

\section{相关研究}
重建式异常检测方法通常假设模型只学习正常样本分布，因此异常区域会产生较大重建误差。这类方法结构清晰，但当重建网络表达能力较强时，异常区域也可能被较好重建，导致异常分数不稳定。DRAEM 在重建网络后加入判别分割网络，通过合成异常生成伪掩码，使模型不仅学习正常图像重建，还直接学习异常区域定位。SimpleNet 通过异常特征模拟学习正常与异常之间的判别边界。EfficientAD 则强调工业部署效率，使用教师-学生网络和自编码器结构获得较快推理速度。

本文复现的 DRAEM-Lite 保留了 DRAEM 的核心思想，即正常样本训练、合成异常构造伪标签、重建网络与分割网络联合优化。为了便于实验，模型采用轻量 U-Net 结构，不依赖预训练主干网络。

\section{模型方法}
Baseline DRAEM-Lite 包含两个 U-Net 风格子网络。给定正常图像 $x$，训练时首先生成合成异常图像 $\tilde{x}$ 和伪掩码 $m$。重建网络 $R$ 输入 $\tilde{x}$，输出重建图像 $\hat{x}=R(\tilde{x})$；判别分割网络 $S$ 将 $\tilde{x}$ 和 $\hat{x}$ 在通道维度拼接，输出异常区域 logits。baseline 损失函数为：
\[
\mathcal{L}_{base}=||\hat{x}-x||_1+
\mathrm{BCE}(S([\tilde{x},\hat{x}]),m).
\]
其中，第一项约束重建图接近正常样本，第二项约束分割网络识别合成异常区域。

改进版 DRAEM-Lite-MS 在 baseline 上加入三点改动。第一，异常合成由简单随机噪声扩展为斑点、矩形污染、线状划痕和稀疏噪点，并融合平滑纹理与局部错位图块，使伪异常更接近工业缺陷形态。第二，在分割损失中加入 Focal loss 和 Dice loss，以缓解异常像素占比低带来的类别不平衡问题。第三，引入多尺度 BCE，对 logits 和 mask 进行不同尺度的平均池化，使模型同时关注局部边缘和整体缺陷区域。测试时，baseline 直接使用分割概率作为异常图；改进版将分割概率与重建残差加权融合，得到最终异常热力图。

\section{实验设置}
实验使用 MVTec AD 数据集中的 bottle、cable 和 hazelnut 三个类别。每个类别的训练集只使用 \texttt{train/good} 中的正常图像；测试集包含 \texttt{test/good} 和不同缺陷类型图像，像素级评价使用 \texttt{ground\_truth} 中的缺陷掩码。输入图像大小设为 $128\times128$，优化器为 AdamW，学习率为 $10^{-3}$，batch size 为 16，baseline 和 improved 均训练 50 个 epoch。

评价指标包括图像级 AUROC、像素级 AUROC 和像素级 Best-F1。图像级 AUROC 衡量模型区分正常图像和异常图像的能力；像素级 AUROC 和 Best-F1 衡量异常区域定位效果。对于缺陷检测任务，像素级指标更能反映模型是否真正找到了缺陷位置。

\section{实验结果与分析}

从结果可以看出，改进版在 bottle 和 cable 两个类别上均取得提升。对于 bottle，图像级 AUROC 从 0.812 提升到 0.952，像素级 AUROC 从 0.488 提升到 0.583，Best-F1 从 0.109 提升到 0.186。该结果说明结构化异常合成和重建残差融合可以帮助模型更稳定地区分瓶身异常。对于 cable，baseline 的图像级 AUROC 只有 0.463，说明其图像级异常评分较弱；改进后提升到 0.575，像素级 AUROC 也从 0.601 提升到 0.744，Best-F1 从 0.057 提升到 0.150，说明多尺度监督对电缆中细长、局部的异常区域更有帮助。

hazelnut 类别的结果与前两个类别不同。baseline 的图像级 AUROC 达到 0.986，像素级 AUROC 为 0.797，Best-F1 为 0.242；改进版反而下降到 0.645、0.786 和 0.226。这说明改进策略并非在所有类别上都稳定有效。原因可能是 hazelnut 的正常纹理和真实缺陷形态与本文合成异常存在差异，改进版在训练中过度依赖合成异常模式，导致图像级异常分数泛化下降。此外，hazelnut 的表面纹理本身较复杂，重建残差可能会把部分正常纹理误判为异常，从而影响最终评分。

总体来看，三类平均图像级 AUROC 从 0.754 降至 0.724，但平均像素级 AUROC 从 0.628 提升到 0.704，平均 Best-F1 从 0.136 提升到 0.187。该结果表明，本文改进方法更主要的收益体现在像素级缺陷定位，而不是所有类别上的图像级判别。对于工业缺陷检测任务而言，定位能力通常比单纯判断是否异常更重要，因此该改进仍具有一定价值。不过，实验也说明合成异常策略需要根据类别纹理和缺陷形态进一步自适应调整。

\section{结论}
本文围绕计算机视觉中的工业缺陷/异常检测任务，从零实现了基于 DRAEM 思路的 DRAEM-Lite baseline，并提出加入结构化异常合成、多尺度监督、Focal loss、Dice loss 和重建残差融合的改进版 DRAEM-Lite-MS。在 MVTec AD 的 bottle、cable 和 hazelnut 三个类别上，改进版在 bottle 与 cable 上提升明显，并提高了三类平均像素级 AUROC 和 Best-F1，说明其对缺陷区域定位有积极作用。但 hazelnut 上图像级 AUROC 明显下降，表明合成异常与真实缺陷分布不一致时，模型仍可能出现泛化不足。后续可进一步引入更接近真实缺陷的异常合成策略，或结合预训练特征方法提升跨类别稳定性。
